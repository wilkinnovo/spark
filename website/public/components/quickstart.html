Get started in 60 seconds

Fastest start — scaffold a ready-to-run app

npx create-spark-html-app yourapp

Sets up Vite, the Spark plugin, and a live reactive welcome screen. Then cd yourapp && npm install && npm run dev.

or wire it up by hand:

1

Install

npm install spark-html
2

Add the Vite plugin & mount

// vite.config.js
import spark from 'spark-html/vite';
export default { plugins: [spark()] };

// main.js
import { mount } from 'spark-html';
mount();
3

Write a component & import it

<!-- components/hello.html -->
<h1>Hello {planet}</h1>
<script> let planet = 'world'; </script>

<!-- index.html -->
<div import="components/hello"></div>
