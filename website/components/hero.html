HTML that reacts.

Spark turns plain .html files into reactive components — markup, script, and style in one file, served straight to the browser.

counter.html
<h2>Count: {count}</h2>
<button onclick={increment}>+1</button>

<script>
  let count = 0;
  function increment() { count++; }
</script>
→

● live — this is the actual component

Count: {count}
